#SuperMercadoProjeto

Projeto simples em Java para gerenciamento de produtos, utilizando conceitos de Programação Orientada a Objetos.

Funcionalidades
	Cadastro de produtos
      Controle de estoque
	Venda de produtos
	Suporte a produtos perecíveis
	Suporte a produtos importados com taxa

Conceitos aplicados
	Encapsulamento
      Herança
	Polimorfismo
	Sobrescrita de métodos

Estrutura
	Produto
	Perecivel
	ProdutoImportado
	Venda
	Supermercado (main)

Como executar
	1.	Abrir no NetBeans ou outra IDE Java
	2.	Compilar o projeto
	3.	Executar a classe Supermercado

Observação

Projeto desenvolvido para fins acadêmicos.
